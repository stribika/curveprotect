WARNING

This is experimental software suite which implements client-side DNSCurve and CurveCP protection.
Use this software very carefully. Software is ready for experimentation, but not yet ready for production use.

Pay special attention to:

Administration interface

Administration interface is not protected against unauthorized local access at this time.
Use it only on Your personal laptop, not on shared multiuser system.

Encrypted filesystem

Directory _CURVEPROTECT_/.keys consist of several key-directories and should be on encrypted filesystem.

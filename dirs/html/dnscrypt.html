DNSCrypt resolver
DNSCrypt resolver is backup DNS resolver running on %TEMPLATE_VARIABLE_ip% port 53 TCP and UDP.
Is dedicated as a last resort DNS resolver when the firewall is transparently redirecting the DNS traffic to a local resolver.
DNSCrypt resolver uses non-statndard port TCP 443 for sending DNSCrypt protected packets.
This will make most firewalls think it's HTTPS traffic and leave it alone.

For example can be configured using public OpenDNS servers:
  • DNSCrypt provider public-key: opendns (key is included)
  • DNSCrypt provider name: 2.dnscrypt-cert.opendns.com
  • DNSCrypt remote resolver IP: 208.67.220.220,208.67.222.222,2620:0:ccc::2,2620:0:ccd::2
